                                                                                         HTML: HyperText Markup Language
*HTML is the standard markup language for creating Web pages,foundation for all websites.
*describes the structure of a Web page
*HTML consists of a series of elements, elements tell the browser how to display the content

ELEMENT
The HTML element is everything from the start tag to the end tag:
<tagname> Content goes here... </tagname>
Ex:<h1>My First Heading</h1>


TAGS:
The <!DOCTYPE html> declaration defines document type ,it is an HTML5 document,not case sensitive,must only appear once,
The <html> element is the root element of an HTML page
The <head> element contains meta information about the HTML page
The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

HTML Attributes:
*All HTML elements can have attributes
*provide additional information,Attributes are always specified in the start tag
Attributes usually come in name/value pairs like: name="value"
*All HTML elements can have attributes
*The src attribute of <img> specifies the path to the image to be displayed
*The width and height attributes of <img> provide size information for images
*The alt attribute of <img> provides an alternate text for an image
*The style attribute is used to add styles to an element, such as color, font, size, and more,The HTML style attribute has the following syntax:
<tagname style="property:value;">
*The lang attribute of the <html> tag declares the language of the Web page,shd be used in html tag
*The title attribute defines some extra information about an element.text over
